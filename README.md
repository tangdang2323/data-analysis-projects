# data-analysis-projects
Учебные проекты по анализу данных
1.
Кейс: Анализ транзакций сети розничных магазинов
Цель проекта: выявить операционные аномалии, оценить влияние маркетинговой акции на прибыль и средний чек, а также визуализировать ключевые показатели бизнеса.
Что делала:
— Провела анализ данных по транзакциям сети розничных магазинов.
— Изучила длительность операций и выявила аномальные значения; интерпретировала их с учётом бизнес-контекста.
— Проанализировала результаты маркетинговой кампании: оценил(а) влияние акции на выручку, прибыль и средний чек.
— Рассчитала коэффициенты прироста и динамику ключевых метрик по периодам.
— Построила интерактивные дашборды и графики в Google Sheets, визуализирующие результаты анализа.

Используемые инструменты: Excel, Google Sheets, функции QUERY, сводные таблицы, визуализации (графики, дашборды).

Результат: подготовлен аналитический отчёт и интерактивный дашборд, демонстрирующие влияние маркетинговой активности на финансовые показатели компании.
