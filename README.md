# retail_analysis

# Анализ транзакций сети розничных магазинов

**Кейс:** Выявление операционных аномалий и оценка эффективности маркетинговой акции.

## Цель проекта
1. Выявить операционные аномалии в транзакциях.
2. Оценить влияние маркетинговой акции на **прибыль**, **выручку** и **средний чек**.
3. Визуализировать ключевые бизнес-показатели.

## Инструменты
- **Excel / Google Sheets** — обработка данных, расчёты
- **Функция QUERY** — агрегация данных
- **Сводные таблицы** — анализ срезов
- **Визуализация** — графики, интерактивные дашборды
